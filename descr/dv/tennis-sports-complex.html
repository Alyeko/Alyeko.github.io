Decision Making System for Tennis Sports Complex

Tools Used: PostgreSQL, QGIS

Description
In this project, a spatial database was created in PostgreSQL from scratch, 7 decisions were made with the data created and a QGIS visualization was finally made.

Processes followed
1. Tables were created together with unique and check constraints
2. Data was inserted into the tables
3. Views were created to aid in simpler querying for decision making
4. The data was then finally visualized in QGIS

Results
A spatial database which contains all the important information about the tennis sports complex, which can be queried to aid in the effective decision making and smooth runnng of the tennis sports complex
