Interactive Cesium Dashboard

Tools Used: PostgreSQL, NodeJS, Bootstrap, Javascript, HTML5, CSS

Description
In this project, in order to visualize and keep record of data from a PostgreSQL spatial database, an interactive dashboard was created with the following components:
a. A cesium map: Shows the locations of the points of the database in the real-world.
b. A bar grpah: Shows the data points and their correspondng rating values.
c. A polar area chart: Also shows the data points and their corresponding rating values, but in a different way from the bar chart.
